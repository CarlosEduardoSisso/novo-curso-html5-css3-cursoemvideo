Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo no meio do par de tag's <p> e </p>.

Se precisar quebrar o texto em qualquer lugar específico,
como aqui,
precisa colocar uma tag <br>, que significa "break row".

Vamos adicionar alguns símbolos adicionais

Algumas marcas:

® para "Marca Registrada";

© para "Copyright";

™ para "Trademark";

Moedas, por exemplo:

€ Euro;

£ Libra;

¥ Yen (moeda japonesa);

¢ Cent (centavo do dólar);

Outros símbolos:

δ Delta Minúsculo;

Δ Delta Maiúsculo;

↑ Seta para cima;

↑ Versão simplificada para o mesmo código;

⇑ Seta dupla para cima;

↓ Seta para baixo;

⇓ Seta dupla para baixo;
