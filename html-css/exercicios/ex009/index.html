Caracteres Mono Espaçados (MONOSPACE)

O comando document.getElementeById('teste') é escrito em JavaScript.

Abaixo, um código do Python:

num = int(input('Digite um número')) if num % 2 == 0: print(f 'O número {num} é PAR.') else: print(f 'O número {num} é ÍMPAR.') print (f 'Fim do Programa')

Veja que, apesar dos caracteres mono espaçados, o código perdeu sua característica sem as indentações próprias.

Abaixo, o mesmo código, agora com as indentações, usando uma tag "PRE":

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f 'O número {num} é PAR.')
    else:
        print(f 'O número {num} é ÍMPAR.')
    print (f 'Fim do Programa')
